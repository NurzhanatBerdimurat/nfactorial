1. UserManager Class:
   - addUser(name): добавляет имя возвращает ID.
   - getUser(user_id): возвращает имя пользователя при вводе ID.
   - deleteUser(user_id): удаляет по ID если успешно True если нет False.
   - findUserByName(name): возвращает список идентификаторов
пользователей с данным именем. Если таких пользователей нет, возвращает
пустой список.

2. Основная функция:
   - используем командную строку для взаимодействия.
   - надо выбрать: add, get, delete, find users, or exit.

### Running the Code:
1. сохранить файл uchet.py.
2. запускаем скрипт в командной строке
